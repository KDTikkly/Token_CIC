# Cicerone (CIC) Token Smart Contract

A customized **ERC-20** token featuring a fixed supply and an automated **1% transaction tax**. This project is structured for flexible deployment using either the Remix IDE ecosystem or local development frameworks like Hardhat.

## üíª IDE Usage & Deployment (Remix IDE)

This project is optimized for **Remix IDE**. It includes helper scripts that automate the deployment process using the artifacts generated by the compiler.

### 1. Preparation

1. Open [Remix IDE](https://remix.ethereum.org).
2. Ensure your file explorer contains:
* `CIC.sol` (The Smart Contract)
* `ethers-lib.ts` & `web3-lib.ts` (The Utility Libraries)
* `deploy_with_ethers.ts` (The Deployment Script)



### 2. Compilation

1. Select the **Solidity Compiler** plugin.
2. Set the compiler version to `0.8.20` (or your preferred 0.8.x version).
3. Click **Compile CIC.sol**. This generates the JSON artifacts in the `browser/artifacts` folder required by the scripts.

### 3. Execution

To deploy the contract using the provided TypeScript scripts:

1. Open `deploy_with_ethers.ts` (or `deploy_with_web3.ts`).
2. **Update the script:** Change the contract name in the `deploy` function from `'Storage'` to `'Cicerone'`.
3. Right-click the script file and select **Run**.
4. The terminal will output the contract address and deployment status.

## ü™ô Tokenomics & Tax Logic

* **Token Name:** Cicerone
* **Ticker:** CIC
* **Total Supply:** 1,000,000,000 (1 Billion)
* **Decimals:** 6
* **Transaction Tax:** **1%**
* **Mechanism:** Every `transfer` and `transferFrom` automatically calculates a 1% fee.
* **Destination:** The tax is sent directly to the `owner` (the address that deployed the contract).
* **Calculation:** $Amount_{Received} = Amount_{Sent} \times 0.99$

For local development or deployment to public testnets (Sepolia/Mainnet), use the following `hardhat.config.js`. It is configured to match the settings expected by your `CIC.sol` file.

```javascript

  solidity: {
    version: "0.8.24", // Supports the ^0.8.0 requirement
    settings: {
      optimizer: {
        enabled: true,
        runs: 200,
      },
    },
  },
  networks: {
    // Local development node
    localhost: {
      url: "http://127.0.0.1:8545"
    },
    // Example Sepolia configuration
    /*
    sepolia: {
      url: "https://eth-sepolia.g.alchemy.com/v2/YOUR_API_KEY",
      accounts: ["0xYOUR_PRIVATE_KEY"]
    }

  },

```

### Deploying via Hardhat CLI

If you move these files to a local Hardhat project:

```bash
# Install dependencies
npm install --save-dev hardhat @nomicfoundation/hardhat-toolbox

# Compile
npx hardhat compile

# Deploy to local network
npx hardhat run scripts/deploy.js --network localhost

```
